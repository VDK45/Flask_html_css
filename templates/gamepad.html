    Кнопки:

    • Управление движения:
      • Button W = UP
      • Button S = DOWN
      • Button A = LEFT
      • Button D = RIGHT
    • Управление действии:
      • Button = SPACES
      • Button = B
      • Button = P
      • Button = N
